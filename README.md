# Aquify Website  

Aquify is a premium water brand website built with **HTML, CSS, and JavaScript**.  

## Features  
- Hero section with animated text  
- Responsive gallery with hover effects  
- Testimonials section with card design  
- Contact form with validation  
- About page with styled content  
- Footer with social media links and back-to-top button  

## How to Run  
1. Clone the repository  
2. Open `index.html` in your browser  
3. Explore the features
